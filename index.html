ECOS DEL SILENCIO

Educational Video Game About Bullying and Cyberbullying

About the Project

ECOS DEL SILENCIO is an educational video game developed by the ECHOMIND team. The project aims to raise awareness about bullying and cyberbullying through interactive storytelling and decision-making.

Gameplay

Victim

Experience bullying situations from the victim's perspective.

Bystander

Decide whether to intervene or remain silent.

Aggressor

Understand the consequences of harmful actions.

Decision System

  • Empathy
  • Stress
  • Support
  • Responsibility
  • Risk

Technologies Used

  • Godot Engine
  • GDScript
  • GitHub
  • Trello
  • Object-Oriented Programming
  • Data Structures

Software Design

The project was developed using Object-Oriented Programming principles. Classes such as Player, Decision, Scenario and GameManager were used to create a modular and scalable architecture.

Testing Results

Five participants tested the game and evaluated usability, interface quality and message understanding. Results showed positive feedback regarding empathy generation and awareness of bullying and cyberbullying.
